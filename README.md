AI/ML Prediction-Based Case Studies

This repository contains machine learning mini projects implemented using Python and Google Colab. 
The projects focus on solving real-world prediction and classification problems using different machine learning algorithms and data visualization techniques.

Implemented Case Studies:

1. Titanic Survival Prediction

Developed a classification model to predict passenger survival in the Titanic dataset using Gaussian Naive Bayes classification. 
The project includes data preprocessing, feature engineering, model training, and confusion matrix visualization.

Technologies Used:
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn
Google Colab
Algorithm Used
Gaussian Naive Bayes
Features
Data cleaning and preprocessing
Missing value handling
Feature engineering
Classification report generation
Confusion matrix visualization

2. Heart Disease Prediction

Developed a machine learning model to predict heart disease using medical parameters such as cholesterol, blood pressure, chest pain type, and heart rate. 
Random Forest classification was used to achieve high prediction accuracy.

Technologies Used:
Python
Pandas
NumPy
Scikit-learn
Matplotlib
Seaborn
Google Colab
Algorithm Used
Random Forest Classifier
Features
Medical dataset preprocessing
Categorical data encoding
Correlation heatmap visualization
Disease prediction
Confusion matrix visualization

3. Crop Recommendation System

Designed a crop recommendation system that predicts suitable crops based on soil nutrients and environmental conditions such as temperature, humidity, rainfall, and pH values.

Results:
Titanic Survival Prediction achieved satisfactory classification accuracy using Gaussian Naive Bayes.
Heart Disease Prediction achieved high accuracy using Random Forest classification.
Crop Recommendation System achieved excellent prediction accuracy with Random Forest classifier.


Conclusion:
These case studies demonstrate the practical implementation of machine learning techniques for solving classification and recommendation problems. 
The projects include data preprocessing, feature engineering, model training, evaluation metrics, and visualization techniques for better understanding of model performance.
